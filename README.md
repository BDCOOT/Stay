# Shop - 개인 프로젝트

## 프로젝트 소개 

&nbsp;전자 상거래 사이트로 회원가입 이후 판매자 신청을 통해 카테고리 분류, 수량, 가격 등을 책정하여 상품을 이미지와 함께 등록하고

일반회원은 장바구니에 담고 문의를 남기거나 구매할 수 있습니다. 구매후에는 리뷰 작성또한 가능합니다.

관리자는 회원 관리 및 공지사항을 작성하거나 상품을 삭제할 수 있습니다.

## 개발 기간 및 환경
개발 기간 : 2024년 1월

language : Java 17

Framework: Springboot(3.2.1)

Database: MySQL

ORM : JPA
